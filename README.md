# CSV Cleaner Web App

A Flask web application that allows users to upload CSV files, clean and process the data, and download a cleaned version with summary statistics.

## Features
- CSV upload via browser
- Backend data validation and cleaning
- Summary statistics displayed on page
- Download cleaned CSV file

## Tech Stack
- Python
- Flask
- HTML

## Usage
1. Install Flask
2. Run `python app.py`
3. Open browser at `http://127.0.0.1:5000`
4. Upload a CSV file

## Use Cases
- Cleaning survey data
- Preparing student or business records
- Removing invalid entries before analysis
